# MelBroadcast Productions — One-page website

## Files
- `index.html` — main page (single-file site)

## Edit
Open `index.html` and replace:
- `mel@melbroadcastproductions.com` with your email
- `+971 XX XXX XXXX` with your phone/WhatsApp
- LinkedIn link

## Deploy (free)
### GitHub Pages
1. Create repo
2. Upload `index.html`
3. Settings → Pages → Deploy from `main` branch

### Netlify
Drag-drop the folder (or zip) into Netlify.

### Cloudflare Pages
Connect repo and deploy.
